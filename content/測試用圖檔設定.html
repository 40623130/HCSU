測試用圖檔設定

測試用球桌圖檔(Onshape)

首先需要簡易球桌與球，如下圖

並將圖檔以stl格式匯出

匯入CoppeliaSim(V-rep)

File  → Import → Mesh

點選Import

拆解組合圖

左鍵選取需拆解之零件

點選右鍵 → Edit → Grouping/Merging → Divide selected shapes

完成後圖檔會從組合拆解為各自零件

更改球體顏色(用於影像辨識)

左鍵點選欲更改之物件 →  點選左側工具欄之放大鏡 →  Adjust color  → Ambient/diffuse component →  使用RGB或HSL調整顏色

自由度設定

新增一個立方體用以固定左右移動軸

新增滑動接頭

固定至相應位置，並更改樹狀資料如下圖

(球桌短邊命名為X、長邊命名為Y)
