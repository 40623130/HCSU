Q&A

如何編輯此網站

請先clone本倉儲

倉儲內有一個資料夾(cmsimde)此時會是空的

再來利用cmd進入本倉儲資料夾並輸入

git submodule update

跑完後資料夾(cmsimde)裡面會有檔案

如果要編輯跟之前不同之處只差在要執行的wsgi.py在資料夾(cmsimde)裡面

所以只要先輸入

cd cmsimde

再輸入

python wsgi.py

之後就可以進行編輯了
